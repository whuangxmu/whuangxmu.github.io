# 三、程序设计题

14. 请用C++语言编程模拟以太网网卡根据目的地址判断一个帧应该被接收。先定义宏 MAC_ADDRESS_LENGTH，再用 typedef 定义 MAC_address 类型和以太网帧的结构体 EthernetFrame，定义本机地址 `MAC_address this_mac_address;`，再书写函数 `int mac_address_match(const struct EthernetFrame *frame);` 判断是否本机地址、多播（多播位为1）或广播地址，返回值 0 为不匹配，1 为匹配。

示例接口与结构（示意）：

```c
#define MAC_ADDRESS_LENGTH 6
typedef unsigned char MAC_address[MAC_ADDRESS_LENGTH];

struct EthernetFrame {
    MAC_address dst;
    MAC_address src;
    unsigned short type;
    unsigned char payload[1500];
};

int mac_address_match(const struct EthernetFrame *frame);
```

```c
    unsigned char src[6];

```
