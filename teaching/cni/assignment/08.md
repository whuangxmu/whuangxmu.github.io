27. Nyquist 采样定理说：对带宽有限的连续信号，采样频率若大于两倍最高有效频率（采样率 > 2 * f_max），则可从样本重建原信号。请用 C++ 编程模拟并验证：当采样频率小于 2*f_max 时无法完整重建。

建议实现要点（可直接在此文件中粘贴或另存为 .cpp）：

- 生成一个已知连续信号，例如单频正弦波 s(t) = sin(2π f0 t)，选择 f0 为基准频率（例如 f0 = 500 Hz）。
- 用两组采样频率分别采样：一组低于 2*f0（例如 1.5*f0），一组高于 2*f0（例如 3*f0）。
- 用 sinc 插值或频域零填充方法对采样点进行重建，并计算重建信号与原始连续信号在重建区间上的均方误差（MSE）。
- 比较两组 MSE，验证高采样率组的误差显著小于低采样率组。

下面给出一个简洁的 C++ 示例骨架（供参考）：

```cpp
#include <cmath>
#include <vector>
#include <iostream>

// 计算 sinc(x) = sin(pi x)/(pi x)
static double sinc(double x) {
    if (std::abs(x) < 1e-12) return 1.0;
    return std::sin(M_PI * x) / (M_PI * x);
}

// 使用 sinc 插值重建点（简单实现，未做效率优化）
std::vector<double> sinc_reconstruct(const std::vector<double>& samples, double fs, const std::vector<double>& t_out) {
    size_t N = samples.size();
    std::vector<double> y(t_out.size(), 0.0);
    for (size_t i = 0; i < t_out.size(); ++i) {
        double t = t_out[i];
        double val = 0.0;
        for (size_t n = 0; n < N; ++n) {
            double tn = n / fs;
            val += samples[n] * sinc((t - tn) * fs);
        }
        y[i] = val;
    }
    return y;
}

int main() {
    // 参数示例
    double f0 = 500.0; // 原始信号频率
    double duration = 0.01; // 10 ms

    // 高采样率组 (3 * f0)
    double fs_high = 3.0 * f0;
    // 低采样率组 (1.5 * f0)
    double fs_low = 1.5 * f0;

    // 生成采样点并重建，计算 MSE（实现细节略，可按需扩展）

    return 0;
}
```
