# 实验目的
通过完成实验，理解网络层和路由的基本原理。掌握路由器配置网络和组网的方法；掌握 IP 协议、IP 地址配置和路由的概念；掌握 IP 协议和路由的基本原理；了解在模拟器下根据教程配置网络的方法。

# 实验要求
使用 Router eSIM v1.1 模拟器来模拟路由器的配置环境；使用 CCNA Network Visualizer 6.0 配置静态路由、动态路由和交换机端口的 VLAN（虚拟局域网）。
